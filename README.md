## 🏫 Сайт центра семейного образования

Полноценное клиент-серверное приложение - мультистраничный сайт-визитка с админ-панелью.

**Мой вклад:**
- Полная frontend-часть на Next.js: адаптивная вёрстка, клиент-серверное взаимодействие, архитектура
- Семантическая разметка, доступность, полная SEO-оптимизация
- Админ-панель с CRUD-операциями
- Backend на Nest.js: PostgreSQL, Redis (хранение сессий), загрузка файлов, авторизация
- Самостоятельный деплой на VPS-сервер

**Стек:** Next.js · Nest.js · TypeScript · PostgreSQL · Redis · Docker · Tailwind CSS

## 🎮 Сервис интернет-услуг

Клиент-серверное приложение — платформа с личным кабинетом, админ-панелью и интеграцией с Telegram.

**Мой вклад:**
- Frontend на React + TypeScript: вёрстка, взаимодействие с API, оптимизация, архитектура - основной фокус в проекте
- Админ-панель, интерфейс личного кабинета
- Участие в backend-разработке на Nest.js (PostgreSQL через Docker, авторизация через Telegram-бота, взаимодействие с пользователями через Telegraph)

**Стек:** React · Nest.js · TypeScript · PostgreSQL · Docker · Tailwind CSS · Telegram Bot API
